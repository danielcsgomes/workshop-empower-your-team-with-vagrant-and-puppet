## Workshop - Empower your team with Vagrant and Puppet

**Vagrant** and **puppet** configuration files for the Workshop at **The Startup Scholarship 2013**

## Requirements
* Virtual Box v4.2.12
* Vagrant 1.2.2

## What will be installed on Virtual Box machine

* Ubuntu 12.04
* NodeJS v0.10.12
* MongoDB 2.4.5
